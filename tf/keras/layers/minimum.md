Functional interface to the `Minimum` layer.

``` python
tf.keras.layers.minimum(
    inputs,
    **kwargs

```

#### Arguments:


* <b>`inputs`</b>: A list of input tensors (at least 2).
* <b>`**kwargs`</b>: Standard layer keyword arguments.


#### Returns:

A tensor, the element-wise minimum of the inputs.
