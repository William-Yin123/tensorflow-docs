Returns the index of the maximum value along an axis.

``` python
tf.keras.backend.argmax(
    x,
    axis=-1
)
```



<!-- Placeholder for "Used in" -->


#### Arguments:


* <b>`x`</b>: Tensor or variable.
* <b>`axis`</b>: axis along which to perform the reduction.


#### Returns:

A tensor.
