## Class `ModeKeys`

Standard names for Estimator model modes.



<!-- Placeholder for "Used in" -->

The following standard keys are defined:

* `TRAIN`: training/fitting mode.
* `EVAL`: testing/evaluation mode.
* `PREDICT`: predication/inference mode.
